AI Medical Diagnosis Tool – Pneumonia Detection

A deep learning-based medical imaging system that detects pneumonia from chest X-ray images using EfficientNet and visualizes model decisions using Grad-CAM.


Github Link!

Overview

The model achieves strong validation performance and provides visual explanations to improve interpretability.

Features

• Detects Pneumonia vs Normal from chest X-rays

• Transfer Learning with EfficientNetB0

• Data Augmentation

• Interactive Web Application

• Explainable AI with Grad-CAM
