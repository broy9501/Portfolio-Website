Unity Racing Game

A fast-paced racing game developed in Unity using C# as part of a university group project. Inspired by the Mario Kart series, the game features power-ups, AI opponents, lap tracking, and arcade-style gameplay.


Github Link! Game Demo!

Overview

This is a 3D kart racing game designed and built in Unity as part of a collaborative group project. My role involved implementing player controls, lap logic, and contributing to race mechanics. The project helped enhance our understanding of Unity’s physics system, real-time game logic, and team-based software development.

Technologies

• Unity Game Engine (3D)

• C# for game scripting

• Unity Physics and Collision System

• Version control via GitHub / Unity Collaborate

• Team collaboration via Microsoft Teams and weekly stand-ups

Features

• Player and AI-controlled karts

• Power ups, checkpoints, and lap counting

• Custom-built track and environment assets

• Responsive UI with race progress indicators

• Arcade-style camera and sound effects
