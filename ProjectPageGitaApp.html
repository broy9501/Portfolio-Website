Gita App with Ask Krishna Chatbot

An AI-powered mobile application built with React Native and RAG (Retrieval-Augmented Generation).


Github Link!

Overview

This mobile allows users to explore the Bhagavad Gita and receive spiritual guidance through an intelligent chatbot.

Technologies

• React Native

• Expo

• Python, Flask, FAISS, Sentence Transformers

• LLM API Integration

• paraphrase-MiniLM-L3-v2 Embeddings

Features

• Browse Bhagavad Gita verses

• Search by keywords

• AI-powered "Ask Krishna" chatbot (RAG-based)

• Fast semantic search using FAISS

• Backend API with Flask
